== Description ==

= Share to Social Bookmarking =
افزونه اشتراک گذاری مطالب در دنبالر، لینک پد، بالاترین، دنباله، کلوب، گوگل پلاس، توئیتر، فیس بوک و ... 

`<?php if (function_exists('share_to_social_bookmarking')){ ?>
	<?= share_to_social_bookmarking() ?>
<?php } ?>`

== Installation ==

1. به برگه 'افزونه‌ها' در مدیریت وردپرس رجوع کرده و بر روی 'افزودن' کلیک کنید
2. 'Share to Social Bookmarking' را جستجو کنید
3. افزونه ذکر شده را نصب کرده و توضیحات صفحه‌ی مدیریت آن را مطالعه کنید
